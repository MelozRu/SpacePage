# Исследование космоса: тайны вселенной

## Описание проекта
Проект представляет собой одностраничный сайт, посвящённый исследованию космоса и вселенной.
Сайт состоит из заголовка, подвала и 9 секций, рассказывающих о методах исследования космоса, 
истории космической гонки, интересных фактах о вселенной и принципах освоения космического пространства.

В проекте использованы различные приёмы в вёрстке **HTML/CSS**, произведена работа с анимацией с помощью **@keyframes** и
осуществлено подключение сторонних медиаресурсов при помощи **@iframes**.

## Используемые технологии

- HTML5
- CSS3
  - Flexbox
  - Positioning
  - Animations
  - Iframe
- БЭМ Nested
